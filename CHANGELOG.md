## 0.3.1 (build 001) — 2026-07-01

첫 배포 — 폴더 스킬을 Claude Code 플러그인으로 패키징.

- **패키징:** `.claude-plugin/plugin.json` + `skills/teachme/` 구조. `/teachme` 명령으로 호출.
- **콘텐츠 유형:** 레퍼런스(Manual) · 실습(Examples) · 서술형 챕터(Chapter) · 문제 풀이/Q&A(Quiz). 주제에 맞게 동적 구성 제안.
- **다국어:** 사용자 요청 언어로 작성(코드는 영어, 고유명사 원형). 언어별 검수, 한국어 최엄격.
- **컬러 테마 6종:** azure(기본)·graphite·ink·teal·amber·indigo (`apply-theme.sh` 배포 시 적용).
- **검색:** 랜딩 전체검색 + 페이지 내 검색.
- **품질 검증:** 멀티에이전트 하네스(내용·언어), SVG 도식화(선택).
- **보안:** 배포 전 시크릿 스캔 관문 + 산출물용 `.gitignore`(.env 등 제외).
- **배포:** 산출물은 비공개 저장소 + 공개 GitHub Pages 기본(무료 계정이면 공개 안내).
- **배포 후 저장소 README 생성:** 배포로 Pages URL이 확정되면 저장소 루트에 라이브 링크 + 설명 + 목차를 담은 `README.md`를 생성(§12-D). docs/ 밖이라 Pages에는 미게시.
